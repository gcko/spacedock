Probe the minimum first-class Pi runtime support needed for Spacedock to pass a basic shared live-workflow baseline.

The intent is to treat Pi as a real runtime target rather than a disguised Claude/Codex path, but to start with the smallest useful proving bar. The research and scope-cutting should happen in ideation, with attention to what the Pi harness actually supports, what extra runtime glue is needed, and which features can remain out of scope for the first slice.
